# VSCODE Template

<br>

## Skill

- include-html (w3_inc.js)
- scss (live-sass)

## Guide

#### ruby install

```bash
$ gen install sass
$ sass --watch public/sass:public/css --style compact
```

#### script load
```html
<script src="path/w3_inc.js"></script>
```

#### include-html 
```html
<div include-html="path/fileName.html"></div>
```
